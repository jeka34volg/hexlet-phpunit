## Addons

Use <http://psysh.org/>

## Setup

```sh
$ git clone git@github.com:jeka34volg/hexlet-phpunit.git

$ cd hexlet-phpunit

$ make install
```

## Run tests

```sh
$ make test
```

## Test Coverage

-   see `phpunit.xml`
-   <https://docs.codeclimate.com/docs/configuring-test-coverage>
